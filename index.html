LyricFinder

Bring real-time lyrics to your Beat Saber experience!

About the Mod

LyricFinder integrates with Genius to fetch lyrics for both official and custom songs in Beat Saber. Enjoy a fully immersive experience with lyrics displayed in sync as you play!

Features

  • Real-time lyrics display for official and custom songs
  • Seamless integration with Genius API
  • In-game overlay to follow the lyrics as you play
  • Easy to set up and customizable

Installation

1. Download the latest version of the mod from the link below.

2. Place the mod's .dll file into your Beat Saber Plugins folder.

3. Launch Beat Saber and start enjoying lyrics in real time!

This mod is not available for download yet. Check back soon for updates!
